# mruby-rack-session-redis

Rack Session Class for mruby

## install by mrbgems

- add conf.gem line to `build_config.rb`

```ruby
MRuby::Build.new do |conf|

    # ... (snip) ...

    conf.gem :github => 'accaman/mruby-rack-session-redis'
end
```
